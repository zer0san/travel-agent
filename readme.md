# 项目介绍

agent项目，智能旅行规划推荐专家。

# 技术栈

- Python
- LangChain
- MCP
- Vue + Vue Router
- ElementPlus

# 使用API

- 高德MCP，获得天气、地图等
- Unsplash，获得相关图片
- 阿里百炼大模型，LLM
